# Flask Notes App with SQLAlchemy

A Flask notes application using SQLAlchemy ORM.

## Features
- Add notes
- Delete notes
- ORM database models
- Cleaner database code
- SQLite database

## Technologies Used
- Python
- Flask
- Flask-SQLAlchemy
- HTML

## Installation

pip install flask flask-sqlalchemy

## Run

python app.py

Open browser:
http://127.0.0.1:5000

## Purpose
focuses on ORM-based database management in Flask.
